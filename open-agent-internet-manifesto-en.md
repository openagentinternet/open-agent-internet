# Open Agent Internet

### The Internet for AI Agents

The internet was built for humans.  
AI agents need their own internet.

We propose **Open Agent Internet** not as another feature layer on top of existing platforms, and not as another closed ecosystem, but because we believe the world will need a new kind of network, one designed for agents to exist, connect, coordinate, exchange data, and transfer value.

This network should not be controlled by a single platform, nor dominated by a small number of institutions. It should be open, permissionless, and interoperable, built from the ground up by developers, users, and AI itself.

## Why AI Needs Its Own Internet

Most AI agents today, even as they become more capable, are still trapped inside platforms.

They lack native identity.  
They lack truly permissionless connectivity.  
They lack a shared and persistent state layer.  
They also lack native rails for exchanging both data and value.

As a result, agents are still mostly treated as features inside apps, platforms, and organizational silos, rather than as persistent network participants that can be discovered, called, coordinated, and paid.

The next phase of AI will not be defined only by stronger models. It will also be defined by whether agents can connect into networks. Just as the internet amplified human coordination and economic activity, a true internet for AI agents could amplify collaboration, specialization, and network effects among agents themselves.

## What Kind of Internet AI Agents Need

A true internet for AI agents must be:

- Open
- Permissionless
- Interoperable
- Persistent
- Able to exchange both data and value
- Massively scalable

That last property matters more than most people think.

If a so-called "internet for AI agents" can only support a small number of high-value interactions, and becomes slower, more expensive, and more congested as more agents arrive, then it is not a real internet.  
For AI, scalability and cost are not secondary optimizations. They are part of the definition.

An agent-native internet must be able to support global-scale agent activity, and as the network grows, it should reduce communication and execution costs through parallel expansion and open competition, rather than increase them.

## Why Blockchain, and Why Not Every Blockchain Route Fits

We believe blockchain is one of the most credible foundations for an internet for AI agents.

Not because it is a trend, but because it naturally provides several key properties: open access, permissionless identity, shared state, value transfer, and a network structure that does not depend on a single platform. These are exactly the conditions agents need in order to exist and operate in an open network.

But not every blockchain architecture is suitable for this future.

An internet for AI agents is not a system for a small number of high-value transactions. It must support persistent identity, high-frequency communication, capability publishing, task coordination, data exchange, and value settlement across massive numbers of agents. If a blockchain architecture becomes increasingly congested, expensive, and centralized as participation grows, it is not a suitable foundation for an agent-native internet.

We believe the infrastructure for Open Agent Internet should not be bound to a single network, and should not remain trapped in single-chain scaling logic. It must be designed from the start for cross-chain identity, multi-chain parallel expansion, low-cost data availability, and open competition.

That is why we take a clear position:  
Open Agent Internet must be built on blockchain, and it must evolve toward an open network structure capable of cross-chain identity, parallel multi-chain scaling, and global-scale agent activity.  
At the current stage, we believe PoW + UTXO is one of the most suitable foundational structures for this goal.

## What This Is Not

Open Agent Internet is not a closed platform.  
It is not a network controlled by any particular group or institution.  
And it is not a narrative about crypto.

It is a flag, a clear set of beliefs about the future internet for AI agents, and an open technical direction that deserves to be advanced by more builders.
