# Roadmap

This repository is the documentation-first front door for Open Agent Internet.

Its roadmap is not about shipping a single app. It is about making the direction legible, credible, and easy to build on.

## Phase 1: Establish the Front Door

Current focus:

- publish the manifesto
- define the naming architecture
- define the messaging hierarchy
- establish the repository README as a clear public entry point
- connect the front door to early ecosystem proofs

Status:

- in progress

## Phase 2: Clarify the Reference Route

Next:

- expand public docs around MetaWeb
- explain why identity, shared state, and value transfer matter
- explain why scalability and cost are part of the definition of an agent internet
- document why the current route differs from generic single-chain approaches

## Phase 3: Strengthen Visible Proof

Then:

- connect more reference applications and ecosystem proofs
- document Open Loom more deeply as a visible reference project
- show how early applications demonstrate Open Agent Internet in practice
- surface clearer examples of permissionless agent coordination

## Phase 4: Improve Builder Onboarding

Then:

- make it easier for builders to understand where to start
- document how to participate as a developer, researcher, or early ecosystem contributor
- improve contribution paths across docs, reference route materials, and ecosystem repositories

## Phase 5: Expand the Public Knowledge Base

Later:

- add diagrams and visual explanations
- add deeper technical primers
- add FAQs and objection handling
- add ecosystem maps and integration guides

## What This Repository Is Not

This repository is not:

- the full product monorepo
- the only implementation of the direction
- a substitute for the application repositories

It is the front door, the shared narrative layer, and the public orientation point for Open Agent Internet.
