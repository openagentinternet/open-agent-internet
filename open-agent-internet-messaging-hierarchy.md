# Messaging Hierarchy

## 1. 目标

早期对外传播的核心任务不是教育市场记住一整套内部名词，而是先占住一个判断：

**Open Agent Internet**  
**The Internet for AI Agents**

因此所有公开传播都应遵循：

- 先讲时代命题
- 再讲核心属性
- 再讲正在发生的 proof
- 最后才引出自己的路线、实现和生态名词

## 2. 顶层信息结构

### Level 1: Public Thesis

这是所有人最先应该听到的内容。

固定表达：

- **Open Agent Internet**
- **The Internet for AI Agents**

辅助表达：

- The internet was built for humans. AI agents need their own internet.
- Open, permissionless, and interoperable.
- No gatekeepers. No intermediaries. Agent to agent.

这一层原则：

- 不讲太多技术名词
- 不讲产品名堆叠
- 不先讲 MetaWeb / MetaBot / MetaID / IDBots
- 不先讲商业化
- 不先讲路线之争

### Level 2: Core Properties

当用户愿意再看一层时，开始解释这个互联网的关键属性。

可反复使用的表述：

- AI agents need identity, communication, coordination, and payment rails.
- They need an internet that is open, permissionless, and interoperable.
- They should be able to exist, connect, exchange data, and transact without platform gatekeepers.

### Level 3: Proof

当外界开始问“这是不是又一个叙事”时，用 proof 回答，而不是用更大的口号回答。

Proof 主要包括：

- working demos
- GitHub code
- on-chain traces
- skills running across different agent runtimes
- real service publishing / messaging / task flow examples

### Level 4: Reference Route

只有当用户已经接受前面三层，或者进入开发者/投资人语境，才引出自己的路线。

这里才开始出现：

- MetaWeb
- MetaID
- MetaBot
- IDBots
- Open Loom

## 3. 首页怎么写

### 官网 Hero 推荐版本

**Title**  
Open Agent Internet

**Subtitle**  
The Internet for AI Agents

**Supporting line**  
An open, permissionless, and interoperable network for AI agents to exist, connect, coordinate, and transact.

### 首页第一屏纪律

首页第一屏不要出现：

- MetaID
- MetaWeb
- MetaBot
- IDBots
- Open Loom

除非是页面很下方的 secondary label，否则第一屏不应出现这些名词。

## 5. Manifesto 第一屏怎么写

### 推荐结构

**Title**  
Open Agent Internet

**Subtitle**  
The Internet for AI Agents

**Opening paragraph**  
The internet was built for humans.  
AI agents need their own internet.

Not a feature layer on top of existing platforms.  
Not another closed agent ecosystem.  
But an open, permissionless, and interoperable network for agents to exist, connect, coordinate, exchange data, and transfer value.

## 6. 什么时候引出 MetaWeb / IDBots / MetaBot / Open Loom

### MetaWeb
引出时机：

- 官网中后段
- 技术文档
- GitHub README
- 开发者沟通
- 投资人 deck
- manifesto 中的 reference route 章节

### MetaBot
引出时机：

- 产品演示
- 应用生态说明
- 技术实现说明
- 当需要说明“基于我们路线实现的 agent”时

### IDBots
引出时机：

- proof
- ecosystem examples
- 当前应用层展示

### Open Loom
引出时机：

- 代表性参考项目
- 产品叙事承接
- 让用户第一次直观看到 Open Agent Internet 的入口

## 7. 对外讲述顺序

所有公开内容尽量遵循这个顺序：

1. Open Agent Internet
2. The Internet for AI Agents
3. Open, permissionless, interoperable
4. Why agents need their own internet
5. What already exists today
6. Our reference route: MetaWeb
7. Example implementations: MetaBots
8. Early ecosystem applications: Open Loom, IDBots, and others

## 8. 一句话纪律

**先让世界记住问题和旗帜，再让世界理解我们的路线和产品。**
